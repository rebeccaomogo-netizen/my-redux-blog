# 🎓 CampusHub - Next.js Redux-Saga Blog

A high-performance, modern Student Information System and Blog Feed built with **Next.js 15**, **Redux-Saga**, and **Tailwind CSS**. This project demonstrates advanced state management, protected routing, and professional UI/UX design.

## 🚀 Links
* **Live Demo:** [https://my-redux-blog.vercel.app](https://my-redux-blog.vercel.app)
* **Repository:** [https://github.com/rebeccaomogo-netizen/my-redux-blog](https://github.com/rebeccaomogo-netizen/my-redux-blog)

---

## 🛠️ Technologies Used
* **Framework:** Next.js 15 (App Router)
* **State Management:** Redux Toolkit & Redux-Saga
* **Styling:** Tailwind CSS (Modern Glassmorphism)
* **Data Fetching:** Axios / DummyJSON API
* **Icons:** Heroicons

---

## ✅ Features Implemented
- [x] **Secure Authentication:** Login flow using Redux-Saga with DummyJSON (User: `emilys`).
- [x] **Protected Routes:** Custom AuthGuard preventing access to unauthorized users.
- [x] **Live Search:** Real-time filtering of blog posts via interactive search bar.
- [x] **Modern UI:** Responsive design with custom gradients, glassmorphism, and custom `rounded-[2.5rem]` corners.
- [x] **Dynamic Post Feed:** Fetching and displaying data from external APIs.
- [x] **Error Handling:** Graceful loading states and error messages.
